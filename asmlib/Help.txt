Данная библиотека включает в себя процедуры работы со строками (str), вводом/выводом (fmt), 
математическими функциями (mth), системными вызовами (sys).
Запускаемый файл - main. Файл помощник при компиляции, компоновки и запуска - Makefile.
Команды Makefile: 
compile - комплирует asm файлы библиотеки
build - компилирует файл main.asm и связывает с объектными файлами библиотеки, полученными при compile
run - запускает файл main 
default - делает build, затем run
Файлы формата inc являются заголовочными файлами, включающие в себя названия процедур.
